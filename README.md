## 環境：Environment
- "core-js": "^3.15.1",
- "nuxt": "^2.15.7",
- "vuetify": "^2.5.8"

## ローカル動作：LocalInstall
```bash
# source clone
$ git clone https://github.com/kametmds/multi-timer.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
